# Day 18 - NLP Project (MeetMux)

## Overview
This project demonstrates:
- Text Cleaning (Tokenization, Stopword Removal, Lemmatization)
- TF-IDF Vectorization
- Sentiment Analysis

## Files
- text_cleaner.py → Cleans raw text
- bio_vectorizer.py → Converts text into numerical features
- sentiment_analysis.py → Analyzes sentiment

## How to Run

```bash
pip install -r requirements.txt
python text_cleaner.py
python bio_vectorizer.py
python sentiment_analysis.py
